
# Final Year Project (FYP) Management System

## Overview
The **Final Year Project (FYP) Management System** is a comprehensive platform designed to streamline the management of final year projects for students, supervisors, and administrators. It provides a centralized hub for project registration, proposal submission, project planning, progression tracking, communication, and support. The system aims to enhance collaboration, efficiency, and transparency throughout the FYP lifecycle.

---

## Features

### 1. **Student Registration**
   - Supports registration for both **individual** and **collaboration projects**.
   - Students can register themselves and choose between self-proposed topics or supervisor-proposed topics.

### 2. **Announcement Board**
   - Admins can post and update announcements for all users.
   - Students and supervisors can view important updates and notifications.

### 3. **Chat Room with Resource Sharing**
   - Real-time communication between students, supervisors, and peers.
   - Supports resource sharing (e.g., documents, links) within the chat.

### 4. **Service & Support**
   - **FAQ**: Answers to frequently asked questions.
   - **Email**: Direct communication with support.
   - **Contact Information**: Access to support contact details.
   - **Submit Feedback**: Users can provide feedback to improve the system.

### 5. **Project Proposal**
   - **Student**:
     - Submit project proposals (individual or supervisor-proposed topics).
     - Await approval from supervisors and admins.
   - **Supervisor**:
     - Approve or reject student project proposals.
     - Propose new topics for students to choose from.
   - **Admin**:
     - Final approval of student project titles.

### 6. **Approval Status Checking**
   - Students can track the status of their project proposals at every stage.

### 7. **Project Planning**
   - Students can plan and organize tasks for the upcoming week.

### 8. **Progression Tracking**
   - Students and supervisors can monitor the progress of the project.
   - Includes updates on completed tasks, milestones, and deadlines.

### 9. **Schedule Meeting**
   - **Student**:
     - Choose available time slots for meetings with supervisors.
     - Submit meeting logs after each session.
   - **Supervisor**:
     - Create and manage available time slots for students.

### 10. **Personal Profile Management**
   - Users (students, supervisors, and admins) can manage their personal profiles.

### 11. **Marksheet**
   - Supervisors can assess and grade students after project completion.

### 12. **User Management**
   - Admins can manage user accounts and permissions.

### Setting Up the Database
1. Within the project file (CIT6224-Web-Dev), there is a database import directory.
2. Enter the directory. 
3. There will be 7 file with .sql extensions, these are the imports that will be needed to setup the project database.
4. Make sure to follow these steps carefully.
	1. Import `users.sql` then
	2. Import `supervisor.sql` then

	4. Import `title_proposed.sql` then
	5. Import `todo.sql` then
	6. Import `todo_joint.sql` then
   7. finally, Import `web_dev_assignment.sql` .
5. Follow this order of importing the database and **VOILAA**!
6. You have **Sucessfully** setup the database.
